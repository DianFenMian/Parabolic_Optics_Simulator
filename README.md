# 抛物线光学性质演示程序

一个交互式的抛物线光学性质演示程序，使用Python和Matplotlib开发，展示抛物线的两大重要光学性质：

1. **平行入射光线聚焦**：平行于抛物线对称轴的入射光线，经抛物线反射后会汇聚到焦点
2. **焦点发射光线平行**：从焦点出发的入射光线，经抛物线反射后会平行于对称轴射出

## 功能特点

### 交互式参数调节
- **焦距调节**：通过滑块可以调节抛物线的焦距（0.5-5.0）
- **光线数量**：控制显示的光线数量（1-15）
- **光线类型切换**：
  - 平行入射光线：从上到下垂直射入抛物线
  - 焦点发射光线：从焦点射向抛物线上的多个点

### 清晰的可视化展示
- 黑色实线表示抛物线
- 红色圆点标记焦点位置
- 黑色虚线表示准线
- 蓝色实线表示入射光线
- 红色实线表示反射光线
- 网格背景，便于观察光线路径

## 安装与运行

1. 确保安装了Python 3.6及以上版本
2. 安装依赖：
   ```bash
   pip install numpy matplotlib
   ```
3. 运行程序：
   ```bash
   python parabola_optics.py
   ```

## 技术实现

- **数学计算**：使用解析几何方法计算抛物线轨迹和光线反射
- **光学反射定律**：入射光线、反射光线和法线在同一平面内，入射角等于反射角
- **交互式GUI**：使用Matplotlib的Widgets模块实现滑块、按钮和单选框
- **中文支持**：正确显示中文标签和标题

## 使用说明

1. 运行程序后，默认显示平行入射光线的反射效果
2. 使用焦距滑块调节抛物线的形状
3. 使用光线数量滑块控制显示的光线数量
4. 点击单选按钮切换光线类型
5. 点击重置按钮恢复初始参数

## 应用场景

- 物理教学：直观展示抛物线的光学性质
- 工程设计：理解抛物面反射镜和天线的工作原理
- 科普演示：向大众展示光学反射现象
